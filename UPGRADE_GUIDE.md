# 治疗师工作量管理系统 - 升级指南

## 版本信息
- **当前版本**: V7.3.0
- **更新日期**: 2026-02-27
- **更新内容**: 成就徽章系统 + 删除功能安全开关 + 自动迁移

## 升级步骤（保留旧数据）

### 方法一：覆盖更新（推荐）

#### 步骤1：备份旧数据
```
1. 进入旧版本目录
2. 复制 instance 文件夹到桌面或其他安全位置
   （这是您的数据库，包含所有工作量记录）
```

#### 步骤2：关闭旧程序
```
1. 如果程序正在运行，先关闭它
2. 可以在任务管理器中确认没有 WorkloadManagement.exe 进程
```

#### 步骤3：部署新版本
```
方式A - 直接替换：
  1. 将新的 WorkloadManagement.exe 复制到旧版本目录
  2. 覆盖旧文件

方式B - 新目录部署：
  1. 解压/复制新版本到新目录
  2. 将备份的 instance 文件夹复制到新版本目录
  3. 覆盖新版本中的 instance 文件夹
```

#### 步骤4：启动新版本
```
1. 双击 WorkloadManagement.exe
2. 等待系统启动（首次启动会自动迁移数据库）
3. 看到 "访问地址: http://localhost:5001" 表示启动成功
```

#### 步骤5：验证
```
1. 打开浏览器，访问 http://localhost:5001
2. 检查原有数据是否完整
3. 查看右侧"成就系统"是否正常显示
4. 进入"设置"→"录入设置"，确认有"删除功能"开关
```

---

### 方法二：全新安装（不保留旧数据）

```
1. 解压新版本到任意目录
2. 双击 WorkloadManagement.exe
3. 系统自动创建新数据库
4. 访问 http://localhost:5001 开始使用
```

## 新功能使用说明

### 1. 成就徽章系统
- 系统根据工作量自动解锁成就
- 成就显示在右侧"成就系统"区域
- 包含17种不同成就（治疗次数、工作量、连续打卡等）

### 2. 等级排行榜
- 显示等级名称（如"康复新星"、"治疗能手"）
- 显示每个治疗师获得的成就徽章

### 3. 删除功能开关
- 默认禁用删除功能（防止误删）
- 需要在"设置"→"录入设置"中手动开启
- 开启后显示所有删除按钮

## 常见问题

### Q1: 升级后原有数据丢失？
**A**: 请确保正确复制了 `instance` 文件夹。这是您的数据库目录。

### Q2: 成就系统显示500错误？
**A**: 重启程序，系统会自动修复数据库。如果仍有问题，请检查控制台输出。

### Q3: 删除按钮不显示？
**A**: 这是正常的。需要进入"设置"→"录入设置"，开启"删除功能"开关。

### Q4: 如何回滚到旧版本？
**A**:
1. 关闭新版本
2. 使用备份的 instance 文件夹
3. 运行旧版本程序
